# restaurant_list
列出餐廳清單的應用程式

## 功能
+ 1.簡易列出餐廳清單

## 相關安裝
1. 開啟terminal(終端機)，開啟作業目錄下執行：
```
git clone https://github.com/rd1123/restaurant_list.git
```
2. 移動至目錄restaurant_list
```
cd path/restaurant_list
```
3. 使用npm安裝相關套件
```
npm install
```
4. 輸入指令啟動本機伺服器
```
npm run dev
```
出現
```
Server is started on http://localhost:3000
```
即成功，並可至http://localhost:3000顯示結果
